# Jimball

> [!NOTE]  
> for WatchFaceFormat variant see [this](../../../wff/jimball)

Simple digital watchface based on [a joker](https://balatromods.miraheze.org/wiki/Cryptid/Jimball) from [Cryptid](https://github.com/SpectralPack/Cryptid).

Supported features:

- background animation
- sound - plays [Funkytown](https://youtu.be/Z6dqIYKIBSU) while not in ambient
